# Workflow Templates

## Private Contract / Due Diligence Review

Flow:

```text
Matter Intake
-> Chronology Builder
-> Issue Spotting
-> Evidence Matrix
-> Source Support Mapping
-> Risk Analysis
-> Draft Red Flag Memo
-> Human Review
-> Audit Log
```

Outputs:

- Matter Profile;
- Timeline;
- Issue Map;
- Evidence Matrix;
- Missing Materials List;
- Draft Red Flag Memo;
- Review Log;
- Audit Pack JSON;
- Feedback Eval Dataset JSON.

Risk posture: do not state final legal conclusions or autonomous professional
advice. Present risk, evidence strength, assumptions, and missing facts for
expert review.

## Compliance Obligation Pack

Flow:

```text
Regulation Intake
-> Obligation Extraction
-> Business Mapping
-> Gap Assessment
-> Risk Scoring
-> Remediation Plan
-> Approval Workflow
-> Audit Pack
```

Outputs:

- Regulation Summary;
- Obligation Register;
- Business Impact Map;
- Gap Analysis;
- Risk Register;
- Remediation Tracker;
- Compliance Impact Memo;
- Audit Pack JSON.

## Audit Evidence Pack

Flow:

```text
Evidence Intake
-> Workpaper Indexing
-> Control Evidence Screening
-> Evidence Matrix
-> Evidence Mapping
-> Risk Materiality
-> Draft Readiness Memo
-> Human Review
-> Audit Log
```

Outputs:

- Audit Profile;
- Workpaper Index;
- Evidence Readiness Dashboard;
- Control Evidence Matrix;
- Evidence Panel;
- Open Question List;
- Draft Readiness Memo;
- Audit Pack JSON;
- Feedback Eval Dataset JSON.
